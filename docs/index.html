Los tres pilares de la web

La Web fue creada por Tim Berners-Lee y se basa en tres tecnologías fundamentales: HTML, HTTP y URL.

Más información sobre la World Wide Web.

HTML (HyperText Markup Language) es el lenguaje de marcado utilizado para crear las páginas web.
